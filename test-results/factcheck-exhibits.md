## Figure 1 (fig:ai-valuations)

Price-dividend ratios (price / trailing 12-month dividends) for five AI stocks (NVDA, MSFT, GOOGL, META, AMZN) versus the rest of the CRSP universe, plotted monthly from 2019 to 2025. The AI line (blue) rises sharply from ~150 to ~400–500 over the sample, while the Non-AI line (red) stays roughly in the 50–100 range.

### Suspicious features

1. **Y-axis cap at 500.** The AI line approaches the top of the plot area around 2024–2025. Code line 124 (`ylim[2] <- min(ylim[2], 500)`) explicitly caps the y-axis for readability.
2. **Sharp AI run-up from ~200 to ~450 during 2023–2025.** Could be an artifact or data error.
3. **Non-AI P/D appears low (~50–80).** Potentially suspicious if the broad market typically has higher P/D.

### Code check

1. **Y-axis cap.** The cap at 500 is a display choice (line 124). It does not alter data; it only prevents extreme outlier months from compressing the visible range. The AI P/D values near 400–500 are consistent with these five stocks having very low dividend yields (AMZN pays no dividend; NVDA and META have minimal yields), which mechanically produces high P/D ratios. Not an error.

2. **Sharp AI run-up.** The code computes P/D as `total_mcap / trail_div` (line 108) after a 365-day trailing window (lines 103–107). The five AI stocks experienced large market-cap increases during 2023–2025 while their aggregate dividends remained small, so a steep P/D rise is expected. The dividend calculation uses `GREATEST((ret - retx) * mcap_lag, 0)` (line 74), which correctly derives dollar dividends from return differentials and avoids split-adjustment issues (documented in code comment, lines 48–49). Not an error.

3. **Non-AI P/D level.** The Non-AI group is the entire CRSP universe minus five stocks. Many CRSP firms are smaller, higher-dividend-yield stocks, so a market-cap-weighted P/D of 50–80 for this broad group is plausible. The code groups correctly via the `LEFT JOIN` on `ai_permnos` (lines 72–76). Not an error.

**Data verification note.** The figure is generated from a live WRDS/CRSP query; no cached data file exists in `/workspace/data/`. The code logic (SQL query, rolling window, aggregation) is verified as correct. The visible patterns (COVID dip in 2020, tech volatility in 2022, AI boom from 2023) are consistent with known market history.

**Caption check.** The paper caption (paper.tex line 48) reads: "Price-dividend ratios for AI stocks (NVDA, MSFT, GOOGL, META, AMZN) versus the rest of the CRSP universe, computed using trailing 12-month dividends." This matches what the code computes.

**Exhibit verdict: PASS** — Code logic is correct, visible features are consistent with expected market patterns, and the caption accurately describes the figure.

## Table 1 (tab:numerical)

2. **Hedging premium reaches 73.4% at p=0.05.** This is a very large premium.

1. **Large divergence at p=0.05.** Manually verified: with β=0.96, γ=3, δ=0.75, the displacement factor δ^(−γ)=2.37 amplifies the singularity payoff for AI stocks (Φ_A≈6.19) far more than for non-AI stocks (Φ_N≈1.13). At p=0.05, the singularity term dominates, pushing V0_A to 26.5. This is mechanically correct from the model formulas (code lines 29–30, 51–52). Not an error.

2. **Large hedging premium.** The hedging premium compares incomplete-market V0_A (which includes the δ^(−γ) displacement hedge) to complete-market V0_A_CM (which uses Φ_A_CM without δ^(−γ), code line 33). At p=0.05, the difference is (26.5−15.3)/15.3×100=73.2≈73.4% (rounding from full precision). Correct.

3. **Non-AI P/D decreasing with p.** In the model, higher p means the pre-singularity discount rate rises because the singularity payoff for non-AI stocks is modest (Φ_N≈1.13, barely above 1).
